\# Zurich Smart City Network



Interactive web map displaying urban sensing infrastructure in Zürich.



\## Description



This project visualizes different types of urban sensor and geospatial infrastructure deployed in Zürich, including:



\- Surveillance cameras

\- Laser scanners

\- PIR + induction sensors

\- Induction loops

\- Pedestrian counting systems

\- Motor vehicle traffic counting systems

\- Experimental chair tracking sensors (GNSS + occupancy)

\- RTK reference stations

\- GNSS reference stations





The map is built using Leaflet and displays GeoJSON datasets.
